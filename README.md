== Installation ==

= Automatic Installation =
1. Login to your WordPress admin area
2. Go to Plugins > Add New
3. Search for "Live TV Streaming Pro"
4. Click install and activate the plugin

= Manual Installation =
1. Download the plugin zip file
2. Login to your WordPress admin area
3. Go to Plugins > Add New > Upload Plugin
4. Browse and select the downloaded zip file
5. Click Install and then Activate

= After Installation =
1. Go to Live TV > Dashboard in your WordPress admin
2. Follow the setup wizard to configure your first channel
3. Use the `[live_tv_player]` shortcode or Gutenberg block to display the player

== Frequently Asked Questions ==

= Does this plugin work with Gutenberg? =
Yes! Live TV Streaming Pro includes a native Gutenberg block that allows you to easily add live TV players to your posts and pages using the block editor.

= Does it work with the Classic Editor? =
Absolutely. The plugin works seamlessly with both Gutenberg and Classic Editor through shortcodes and visual controls.

= What video formats are supported? =
The plugin supports HLS (.m3u8), MP4, WebM, OGG, and other HTML5 video formats. HLS is recommended for live streaming.

= Can I import M3U playlists? =
Yes, you can import M3U playlists from files or URLs with automatic category mapping and duplicate detection.

= Is Google Cast supported? =
Yes, the plugin includes Google Cast integration for streaming to Chromecast and other Cast-enabled devices.

= Does it work on mobile devices? =
Yes, the plugin is fully responsive and optimized for mobile devices with touch gesture support.

= Can I customize the player appearance? =
Yes, you can customize colors, themes, logos, and use custom CSS for complete styling control.
